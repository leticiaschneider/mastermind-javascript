How to Play Mastermind

  • Guess the secret code.
  • The code consists of a sequence of colored pegs.
  • After each guess, you'll receive feedback:
    • Black peg: Correct color in the correct position.
    • White peg: Correct color in the wrong position.
  • Use the feedback to refine your guesses.
  • You win if you guess the code within the allowed attempts!
